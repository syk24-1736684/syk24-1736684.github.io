What's CSS?

CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media

CSS saves a lot of work. It can control the layout of multiple web pages all at once

What's a property?

The properties state what sort of look and feel you want to apply. For example you use the property "background-color" to apply a background color inside a tag.

What's a value?

The values say what to do with the property. For example you can enter background-color:red to say that you want the background color to be red

How do you say "this property has this value"?

a colon (:) separates the property from the value.

When we have more than one property:value for a tag, we separate the properties with a semi-colon (;)

How do you combine two or more property:value pairs together

You apply one or more property:value pairs to a tag to "style" that tag. Usually, all the property:value pairs are kept outside the HTML

How do you put the property:value pairs inside an HTML tag?

To add a style to a tag, we simply type inside it and put the property:value pairs inside the quotes.
